protected MigrationPluginManagerInterface getMigrationPluginManager()

Returns the migration plugin manager service.

This service cannot be injected as the 'migrate' module might not be enabled and will throw the following exception: > Symfony\Component\DependencyInjection\Exception\ServiceNotFoundException > The service "migrate_runner.commands" has a dependency on a > non-existent service "plugin.manager.migration". Unfortunately, we cannot avoid the class instantiation, via an annotation (as @validate-module-enabled for methods), if a specific module is not installed. Open a followup to tackle this issue.
