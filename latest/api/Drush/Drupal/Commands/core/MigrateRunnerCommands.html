int $sleep
bool $port
bool $browser

Return Value

bool

TRUE if browser was opened. FALSE if browser was disabled by the user or a default browser could not be found.

Returns the migration plugin manager service.

This service cannot be injected as the 'migrate' module might not be enabled and will throw the following exception: > Symfony\Component\DependencyInjection\Exception\ServiceNotFoundException > The service "migrate_runner.commands" has a dependency on a > non-existent service "plugin.manager.migration".
