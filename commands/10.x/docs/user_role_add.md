# user:role:add

Add a role to the specified user accounts.

#### Examples

- <code>drush user-add-role "power user" user3</code>. Add the "power user" role to user3

#### Arguments

- **role**. The name of the role to add.
- **names**. A comma delimited list of user names.

#### Aliases

- urol
- user-add-role
